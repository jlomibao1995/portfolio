Home Nventory Application

This web-based application was designed to allow users to log in to their account and manage their home inventory

System Specifications

  • Back end implemented as REST API with Spring Boot Framework
  • Database made with MySQL
  • Application connection through EclipseLink JPA
  • Front end built with Bootstrap and Angular Framework
  • Payment processing integrated with Paypal REST SDK
  • Development done through VS Code IDE
  • Deployed through AWS EC2
