Parkland Preschool Project

The Parkland Preschool website is a web-based service that allows users to access information and manage registration options related to students who are or may be attending Parkland Preschool. The website is intended to provide guardians with a modern enrollment experience and to eliminate the administrative overhead caused by maintaining a hardcopy system.

System Specifications

  • Implemented with Java Servlets and JSP
  • Database made with MySQL
  • Application connection through EclipseLink JPA
  • Front end built with Bootstrap and WebFlow
  • Payment processing integrated with Paypal REST SDK
  • Development done through NetBeans API
  • Used Apache Tomcat as web server engine
  • Deployed through AWS EC2

Website

Admin Functionality

Manage Registrations

An admin is able to view, accept, deny, deactivate, activate, and delete preschool registrations

Manage Classes

An admin is able to perform CRUD operations as well as manage waitlists for classes

View Payments

An admin can view payments in the system

Print Registration

An admin can print a registration form with the information filled in

Print Reports

An admin can print reports for accounts, registrations, classes, and payments

Guardian Functionality

Manage Children

A guardian can add, update, and delete child information from their account

Register Child

A guardian can register their child to a class

Process Payments

A guardian can pay for their registration fees via PayPal
